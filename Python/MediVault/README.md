# MediVault: Medication Conflict Registry

A Python-based safety tracking application designed to log medical prescriptions and automatically flag potential Adverse Drug Interactions (ADIs) using a centralized conflict registry. This is my first-ever software development project, built with the assistance of AI to help navigate engineering concepts and implement structured codebase logic.

---

## 🛠️ Technical Implementation

* **Relational Logic:** Implements programmatic evaluation structures to compare active medications against a curated interaction database.
* **Data Validation:** Employs input verification logic to ensure medication details are recorded accurately without logging conflicts.
* **Workspace Security:** Built as an isolated system module to safeguard tracking records.
